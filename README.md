# Movie Ratings Analysis (R)

## Project Overview
This project performs exploratory data analysis on a small movie dataset using R. The goal is to analyze genre-wise average ratings and visualize rating trends.

## Tools Used
- R
- dplyr
- ggplot2
- R Markdown

## Analysis Performed
- Created structured movie dataset
- Performed genre-wise aggregation using group_by() and summarise()
- Calculated average ratings per genre
- Identified top-rated movie
- Visualized average ratings using bar chart
- Visualized rating distribution using histogram

## Key Insights
- Sci-Fi genre had the highest average rating.
- Interstellar was the top-rated movie.
- Most movie ratings fall between 7.5 and 9.5.

## Skills Demonstrated
- Data manipulation using dplyr
- Group-wise aggregation
- Sorting and filtering
- Data visualization using ggplot2
- Report writing using R Markdown
